## Как запустить:
>  1. Создаем виртуальное окружение
```python
python3 -m venv env
```
>  2. Активируем
```python
source env/bin/activate
```
> 3. Скачиваем нужные зависимости
```python
pip3 install -r requirements.txt 
```
> 4. Запускаем сервер
```python
python3 manage.py runserver
```
